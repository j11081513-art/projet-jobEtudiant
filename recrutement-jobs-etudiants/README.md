# 🎓 JobEtudiant — Plateforme de Recrutement Jobs à Temps Partiel

> Projet réalisé dans le cadre du cours **Configuration Git & GitHub** 

---

## 📋 Description

**JobEtudiant** est une plateforme web de mise en relation entre :
- **Étudiants** à la recherche de jobs à temps partiel (week-end, soirs, vacances)
- **Employeurs** souhaitant recruter des profils étudiants motivés

---

## 🗂️ Structure du projet

```
recrutement-jobs-etudiants/
│
├── index.html              → Page d'accueil (Hero, Catégories, Comment ça marche)
├── css/
│   └── style.css           → Feuille de style globale
├── js/
│   └── main.js             → Scripts JavaScript (navigation, filtres, formulaire)
├── pages/
│   ├── offres.html         → Liste des offres d'emploi avec filtres
│   ├── inscription.html    → Formulaire d'inscription (étudiant & employeur)
│   └── employeurs.html     → Espace dédié aux employeurs
└── README.md               → Ce fichier
```

---

## 🚀 Fonctionnalités

- 🏠 **Page d'accueil** avec statistiques et catégories par secteur
- 🔎 **Recherche d'offres** avec filtres (secteur, salaire, ville, contrat)
- 📝 **Formulaire d'inscription** double profil (étudiant / employeur)
- 📱 **Design responsive** adapté mobile et desktop
- ✨ **Animations** et micro-interactions modernes

---

## 🛠️ Technologies utilisées

| Tech | Usage |

## ▶️ Lancer le projet

Aucune installation requise. Il suffit d'ouvrir `index.html` dans un navigateur.

```bash
# Cloner le dépôt
git clone https://github.com/votre-username/recrutement-jobs-etudiants.git

# Ouvrir le projet
cd recrutement-jobs-etudiants
open index.html   # macOS
start index.html  # Windows
```
